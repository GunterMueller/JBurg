JBurg References

  • Engineering a Simple, Efficient Code Generator Generator
    Christopher W. Fraser, David R. Hanson, and Todd A. Proebsting
    This reference describes iburg, a BURG that generates back ends for the lcc compiler. JBurg's pattern-matching facilities were based on a close reading of this paper.

  • One-Pass, Optimal Tree Parsing - With Or Without Trees
    Todd A. Proebsting and Benjamin R. Whaley
    An extended discussion of BURS systems that use the "Principle of Non-Interference" to label and reduce subtrees in a single pass.

  • A Study on Generating an Efficient Bottom-up Tree Rewrite Machine for JBurg
    Dr. KyungWoo Kang
    Dr. Kang proposes an analytical technique which would allow JBurg to do as much dynamic programming as possible at compiler-compile time.
    Further investigation and implementation is scheduled for JBurg 2.0.

  • A Tutorial on Dynamic Programming
    Michael A. Trick
    A good introduction to dynamic programming.
