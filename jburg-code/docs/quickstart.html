Quick Start

  1. Download the Source Distribution of JBurg, or check out the source: hg clone http://hg.code.sf.net/p/jburg/code jburg-code
  2. Third party tools required:
    • ANTLR 3.x
      Add antlr.jar's path to build.properties, e.g.,
      antlr.jar=${user.home}/tools/antlr-3.4-complete.jar
  3. ant in the jburg directory builds JBurg and runs first-level acceptance tests.
    These tests are located in the tests/BurmTest directory; they test most features using a simple self-verifying test harness.
